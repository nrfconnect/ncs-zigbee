Libraries for Zigbee¶

Zigbee is a portable, low-power software networking protocol that provides connectivity over an 802.15.4-based mesh network.

Zigbee libraries are a set of components that can be directly reused in Zigbee applications. They implement the default behavior of a Zigbee device and are used in the Samples.
