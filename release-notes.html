All the notable changes to the Zigbee add-on for the nRF Connect SDK are listed here. See also the Release notes for the nRF Connect SDK and the ZBOSS stack changelog

Note

This is an experimental release of Zigbee support for the nRF54L Series.

The Zigbee add-on v0.2.0 is compatible with nRF Connect SDK v2.8.0 and uses the ZBOSS stack version 4.1.4.2. For a full list of Zigbee add-on, related nRF Connect SDK and ZBOSS stack and NCP host package versions, view the following table:

Zigbee add-on v0.2.0 - 28/11/2024

This is an experimental release.

  • Added:

    • The NCP sample.

    • The ZBOSS NCP Host package v3.0.0.

  • Updated the documentation with small improvements.

Zigbee add-on v0.1.0 - 15/11/2024

Initial release.

  • Added:

    • Experimental support for the ZBOSS R23 stack for the nRF54L15 SoC. Released Zigbee library is not certified.

    • Basic samples:

      • Light bulb

      • Light switch

      • Network coordinator

      • Shell

      • Template
