# Zigbee R23 add-on for nRF Connect SDK

The Zigbee R23 add-on for the nRF Connect SDK provides support for developing Zigbee applications based on the third-party precompiled ZBOSS stack. It supports Nordic Semiconductorâ€™s nRF54L Series, nRF5340 and nRF52840 SoCs. This stack is included as the ZBOSS Zigbee R23 stack library. In combination with the nRF Connect SDK, the Zigbee R23 add-on allows for development of low-power connected solutions.

## Getting started
To get started with Zigbee R23 add-on for nRF Connect SDK, follow [documentation](https://docs.nordicsemi.com/bundle/addon-zigbee-r23-latest/page/index.html).
